# CS - Ticket System

 - What is JSX, and why is it used?

 Ans: JSX হল JavaScript XML যা React-এ ব্যবহৃত একটি সিনট্যাক্স এক্সটেনশন যা ডেভেলপারদের জাভাস্ক্রিপ্টের ভিতরে HTML-এর মতো কোড লিখতে দেয় যাতে সহজেই ইউজার ইন্টারফেস তৈরি করা যায়।

 - What is the difference between State and Props?

 Ans: প্রপস হলো একটি প্যারেন্ট কম্পোনেন্ট থেকে একটি চাইল্ড কম্পোনেন্টে পাঠানো ডেটা। State হলো এমন ডেটা যা একটি কম্পোনেন্টের ভিতরের ডেটা জমা রাখে এবং সময়ের সাথে সাথে পরিবর্তিত হতে পারে। যখন State পরিবর্তন হয়, তখন কম্পোনেন্টে ্আবার রেন্ডার হয়।

 - What is the useState hook, and how does it work?

 Ans: useState হলো একটি React হুক যা একটি কম্পোনেন্টে ডেটা জমা রাখে এবং আপডেট করতে দেয়।

 - How can you share state between components in React?

 Ans: React-এ, স্টেটকে কম্পোনেন্টর মধ্যে share করা যেতে পারে, স্টেটটিকে তাদের নিকটতম সাধারণ প্যারেন্টে তুলে এবং ডেটা বা আপডেট ফাংশনগুলিকে প্রপস হিসেবে পাস করে।

 - How is event handling done in React?

 Ans: ইভেন্ট হ্যান্ডলিং বলতে ব্যবহারকারীর ক্রিয়া যেমন ক্লিক, টাইপিং বা মাউসের নড়াচড়ার প্রতি সাড়া দেওয়া বোঝায়।
